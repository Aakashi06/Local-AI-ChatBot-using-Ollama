# 🤖 Local AI Chatbot using Ollama

This project demonstrates how to run a local LLM (Gemma / Phi3) using Ollama and build a simple conversational chatbot in Python.

## 🚀 Features

* Runs completely locally (no API cost)
* Supports multi-turn conversations
* Uses Ollama models like `gemma3:4b` and `phi3`
* Beginner-friendly implementation

## 🛠️ Setup

```bash
pip install ollama
ollama pull phi3
```

Run the chatbot:

```bash
python package.py
```

## 💡 Tech Stack

* Python
* Ollama (Local LLM runtime)
